# Life - An AU/VST3 plugin to add life to digital tracks

## Idea

* This plugin is meant to be added to a bus, either mix or master
* It contains a number of effects run in series
    1. Input transformer - An input transformer like on an old mixer. Controls (on/off)
    2. Noise - Adds analog noise floor to the track, like vintage equipment. Controls (level)
    3. THD - Total Harmonic Distortion. Inherent to older analog gear. Controls (level - drives more the higher)
    4. Saturation - Heavier, emulating vintage tube distortion. Controls (level - drives more the higher)
    5. Output transformer - See #1
* A "wide" toggle. When engaged processes L/R seperately applying subtle randomness to each to widen the sound
* Horizontal layout [IT] -> [N] -> [THD] -> [S] -> [OT] wide control can be at the bottom center
* Same look and feel as Loc-Box, Dist308 (../../plugins)
